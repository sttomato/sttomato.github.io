python常用的训练数据的格式读写汇总

HDF5

HDF5的特点：

  • 易于读取
  • 类似于mat数据，但数据压缩性能更强
  • 需要全部读进内存里，故HDF5文件大小不能超过内存，可以分成多个HDF5文件，将HDF5子文件路径写入txt中.
  • I/O速率不如LMDB.
